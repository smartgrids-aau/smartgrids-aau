# Smart Grids AAU

## Swarm Robotics and Bio-inspired Simulation

* [Environment Mapping with a Swarm of Mechalino Robots](https://github.com/smartgrids-aau/Environment-Mapping-with-a-Swarm-of-Mechalino-Robots)
* [Mechalino Arena](https://github.com/smartgrids-aau/Mechalino_Arena) — Implementation of the SCPP algorithm on real educational robots
* [Swarm Coverage Exploration](https://github.com/smartgrids-aau/Swarm-Coverage-Exploration) — Swarm-based algorithm for robust coverage of multi-robot systems in unknown, obstacle-rich environments
* [SwarmFabSim ABCE](https://github.com/smartgrids-aau/SwarmFabSim_ABCE) — SwarmFab simulator for bottom-up, entropy-regulated job shop scheduling using Artificial Bee Colony algorithm
* [Environmental Complexity Criterion](https://github.com/smartgrids-aau/environmental_complexity_criterion) — Implementation of multi-agent coverage path planning algorithms using the Mesa framework
* [SISMO](https://github.com/smartgrids-aau/SISMO) — Simulation of slime mold (*Physarum polycephalum*) foraging behaviour, using A* to find shortest paths between food sources

## Network Simulation

* [EvoTopo](https://github.com/smartgrids-aau/EvoTopo) — Evolving realistic topologies for wireless mesh network simulation
* [JProwler](https://github.com/smartgrids-aau/jprowler) — Java Probabilistic Wireless Network Simulator

## NILM (Non-Intrusive Load Monitoring)

* [YoMoPie and NILM](https://github.com/smartgrids-aau/YoMoPie_and_NILM) — Demonstration of a simple onboard NILM approach with a self-developed metering shield

## Evolutionary Algorithms

* [Frevo](https://github.com/smartgrids-aau/Frevo) — Open-source Java framework for evolutionary design and optimization, with componentwise separation of problem definition, solution representation, and optimization method
* [LabGenAlg](https://github.com/smartgrids-aau/LabGenAlg) — Labyrinth-generating algorithms
